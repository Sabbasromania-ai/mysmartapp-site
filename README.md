# SmartApps Portfolio

## Για να αλλάξεις περιεχόμενο
Άνοιξε μόνο το αρχείο: `src/config.ts`
- Βάλε το email σου
- Βάλε τα App Store / Play Store links
- Πρόσθεσε/αλλαξε apps

## Deploy στο Vercel

1. Ανέβασε το project στο GitHub
2. Πήγαινε στο vercel.com → New Project
3. Import το repo
4. Vercel το κάνει build αυτόματα (framework: Vite)

## Local dev
```bash
npm install
npm run dev
```
